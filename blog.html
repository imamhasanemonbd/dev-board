Question-1 : What are the different ways to select an element in the DOM?

There are Many ways we can select the element, for example, getElementByID - it's select an element with a specific id , getElementByClassName - it select any element with the class name , getElementByTagName - this select element with tag name like div, h1, p, etc . we also can select element with query selector quaryselectorall, etc.
